
# Climatic

Um porjeto que mostra a previsão do tempo




## Rodando localmente

Clone o projeto

```bash
  git clone https://link-para-o-projeto
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  yarn 
```

Inicie o servidor

```bash
 yarn dev
```


## Stack utilizada

**Front-end:** React, TailwindCSS, MaterialUI
